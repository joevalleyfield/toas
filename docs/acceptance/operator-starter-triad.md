# Operator Starter Triad (Seed Scenarios)

This triad captures low-friction, high-signal operator scenarios that can grow into broader acceptance coverage.

## Primary seed (start here)

1. `/help config` discoverability
- Goal: operator can discover config-oriented help from first principles.
- Why first: minimal setup, deterministic output shape, strong operator value.

## Follow-on candidates

1. `/config values <key>` categorical enumeration
- Goal: operator can query allowed categorical values directly.

1. Slash extraction safety (column-1 only)
- Goal: indented/in-text slash samples stay inert; only column-1 slash commands execute.

## Notes

- Treat `/help config` as the base scaffold for acceptance structure.
- Expand breadth from the same scaffold rather than building synthetic long narratives first.
