# Acceptance Scenarios

This folder holds long-form acceptance scenario specifications.

Executable acceptance artifacts live under `tests/acceptance/`.

Boundary:
- Acceptance specs and executable assertions belong under `docs/acceptance/` and `tests/acceptance/`.
- Operator-process/runbook evolution belongs under `docs/acceptance/spikes/` and task lane `487` (cadence/scorecard), not as acceptance-exit blockers.
